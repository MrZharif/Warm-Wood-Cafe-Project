# Warm-Wood-Cafe-Project
 Proyek ini berfokus pada pembangunan fitur fungsional seperti Keranjang Belanja (Add to Cart) real-time dan simulasi Status Pesanan dinamis ('Masak' -> 'Siap') untuk memberikan pengalaman pengguna (UX) yang menarik. Proyek ini berhasil meningkatkan pemahaman solid tentang implementasi logika dasar JavaScript dan praktik penulisan kode yang bersih.
